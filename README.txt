Инструкция

1. Использовать OpenServer + Chrome последней версии.
2. Создать в папке domains домен и скинуть туда скачанный проект с GitHub.
3. В файле connection_bd.php следует заменить значения хоста, пользователя и пароля на свои.
4. Импортировать файл future_test.sql в phpMyAdmin (добавляем базу данных).
5. Открыть сайт через OpenServer.
